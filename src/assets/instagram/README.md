# Galeria do Instagram (estática / curada)

Coloque aqui as fotos baixadas do Instagram da academia.

## Como adicionar fotos
1. Salve as imagens nesta pasta (`src/assets/instagram/`).
2. Use nomes simples e sequenciais: `1.jpg`, `2.jpg`, `3.jpg`, ... (JPG ou PNG).
3. Liste cada uma em `posts.json` (veja o exemplo lá). O campo `caption` é a legenda
   e `link` é o link do post no Instagram (opcional — se vazio, abre o perfil).
4. Avise "tá na pasta" que eu ligo a galeria da landing nesses arquivos.

## Dicas
- Imagens quadradas (1:1) ficam idênticas ao feed do Instagram. ~1080x1080px é ótimo.
- 6 a 9 fotos deixam o grid bonito (3 colunas).
- Quanto maior a resolução, melhor — o navegador reduz automaticamente.
