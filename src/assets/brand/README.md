# Identidade da marca

Coloque aqui os arquivos da marca para eu aplicar a identidade real:

- `logo.png` (ou `logo.svg`) — logo principal. De preferência com fundo transparente.
- `logo-claro.png` — versão para fundo escuro (opcional).
- `hero.jpg` — foto forte para o topo da landing (treino, tatame, equipe).
- `sobre.jpg` — foto para a seção "Sobre" (opcional).

Depois é só avisar "tá na pasta" que eu puxo logo, hero e extraio a paleta de cores do logo.
